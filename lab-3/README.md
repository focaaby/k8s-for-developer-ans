# Service, Ingress

## Service

1. 建立 Service `nginx-ervice`。

```
$ kubectl apply -f ./lab-3/nginx-svc.yaml
service/nginx-service created
```

2. 觀察 Service `nginx-ervice`。

```
## What is Endpoints
$ kubectl describe svc nginx-service

$ kubectl get svc nginx-service -o wide
```
